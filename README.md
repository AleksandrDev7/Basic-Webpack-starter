# Basic Webpack for Frontend


The project has the following features:

- Webpack v. 5
- ES6 Support via babel (v. 7.18);
- Sass loader;
- url loader;
- Webpack dev server;
- Stylelint;
- JS Linting via eslint;


## Installation

```
npm install
```

## Start Dev Server

```
npm start
```

## Build Prod Version

```
npm run build
```

### The project is production ready.
